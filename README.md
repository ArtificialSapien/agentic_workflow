# Agentic Workflow

## Dependencies

Working with python version
* 3.11.10
* 3.12.7

## Quickstart

Install packages:
```bash
pip install -r requirements.txt
```

In root directory of this repo run:

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000`
```

In your browser open: http://127.0.0.1:8000/docs
